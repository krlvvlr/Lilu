# Lilu

AI-powered Telegram chatbot built with Aiogram and Grok.

## Features

- AI conversations
- Image analysis
- Long-term memory
- Telegram Stars payments
- Referral system
- SQLite database
- Premium access management

## Tech Stack

- Python
- Aiogram 3
- xAI (Grok)
- SQLite
- Pillow
- python-dotenv

## Configuration

Copy `.env.example` to `.env` and configure your API credentials.

## Disclaimer

The personality prompt is fully customizable and provided as an example configuration.
