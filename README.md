# Lilu

AI-powered Telegram chatbot built with Aiogram and Grok.

## Features

- AI conversations
- Long-term memory
- Image analysis
- Image generation
- Telegram Stars payments
- Referral system
- SQLite database

## Tech Stack

- Python
- Aiogram 3
- xAI (Grok)
- SQLite
- Pillow

## Configuration

Copy `.env.example` to `.env` and fill in your credentials.

## Disclaimer

The personality prompt is fully customizable and intended as an example configuration.
